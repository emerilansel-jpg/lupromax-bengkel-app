# Lupromax Bengkel Rewards

Proposal landing page untuk Lupromax x Pesat.AI — loyalty & gamification platform untuk bengkel partner.

## Pages

| Page | Target | URL |
|------|--------|-----|
| Client Page | Tim internal Lupromax | https://apps.pesat.ai/client/lupromax/bengkel-app/ |
| Landing Page | Owner bengkel calon mitra | https://apps.pesat.ai/client/lupromax/bengkel-app/landing-page/ |

## Tech Stack

- Vite + React 19 + TypeScript
- Tailwind CSS v3.4.17
- GSAP + ScrollTrigger
- lucide-react

## Quick Start

```bash
npm install
npm run build
```

## Deploy

```bash
# Build
npm run build

# Landing page must be manually copied to dist/ after build
# Then deploy dist/ to VPS
```
